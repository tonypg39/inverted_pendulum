English:
     1-Open the agents_server folder with VsCode or any text editor you want and run the python3 agent_serve.py scripts
     2-Open the folder InvertedPendulum with VsCode and run ng serve
     3-Open a Browser in localhost: 4200
     -------------------------------------------------- --------------------------------
     In case of step 2 error, you need to install the angular pluggin
     run npm install (connected to the internet) and repeat the process from step 2
